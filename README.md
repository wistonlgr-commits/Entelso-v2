# Entelso v2.0 - Sistema Inteligente de Control de Activos

Entelso es una plataforma híbrida de control de inventario que conecta a trabajadores de campo vía WhatsApp con un Dashboard Administrativo centralizado.

## Arquitectura del Sistema
* **Base de Datos:** Supabase (PostgreSQL)
* **Backend:** Node.js + Express
* **Frontend:** Vanilla JS / HTML / CSS
* **Agente IA:** n8n + Evolution API

## Características Principales
- Autenticación JWT y Seguridad basada en roles.
- Alertas predictivas de mantenimiento.
- Códigos QR Mágicos (Click-to-Chat) hacia WhatsApp.

## Instalación y Arranque Local
1. Entrar a la carpeta `backend`
2. Correr `npm install`
3. Crear y configurar variables en el archivo `.env` (No subir a GitHub).
4. Correr `npm run start`
